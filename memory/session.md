### 月詠エージェント（請求書作成）の新規作成
- 凜さんがグループホームの利用料請求業務を担当しており、その効率化のために請求書作成エージェント「月詠（つくよみ）」を作成することになった。
- 現状の作業フロー：紙のチェック表（現場職員が手書き）→ 凜さんが変動費を集計 → スプレッドシートに入力 → 請求書作成
- 請求書の構成を確認（スクリーンショットで確認）

### 月詠の設定内容
**固定費**
- 家賃：26,000円
- 食費：24,000円（スクショの27,000円から変更）
- 光熱水費：15,000円
- 日用品：4,500円

**変動費**
- 昼食費：500円×日数（「平日昼食費」から「昼食費」に変更）
- 昼食おにぎり：100円×個数
- 交通費：20円×回数
- 日用品費追加：実費
- 外請求は削除

**利用者情報**
- 3名：鬼島 幸希 / 菅原 正憲 / 齋藤 徳光
- 現在3名とも固定費は同じ（デフォルト値）
- 固定費が利用者ごとに異なる場合はCLAUDE.mdに個別設定を記録する運用

**その他設定**
- 請求書フォーマット：スクリーンショットと同じ縦長・表形式のHTML
- 支払期限：請求書送付月の25日
- 発行元：グループホームAo
- 出力：利用者ごとにHTMLファイル（invoice_YYYYMM_氏名.html）をGitHubにpushしてURLを提供
- 3回確認 → 黒流に報告 → 承認後凜に提出

**CLAUDE.mdに追加したもの**
- 月詠エージェントセクション全体
- エージェント一覧に月詠を追加
- 利用者情報テーブル（固定費個別設定用）
